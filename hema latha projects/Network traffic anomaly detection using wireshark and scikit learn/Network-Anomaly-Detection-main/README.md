# Network Traffic Anomaly Detection

## Tools Used
- Python
- Wireshark
- Scikit-learn
- Pandas
- Matplotlib
- Tkinter

## Steps
1. Capture network packets using Wireshark.
2. Export packets to CSV.
3. Preprocess the data.
4. Train an Isolation Forest model.
5. Detect anomalous packets.
6. Visualize the results.

## Output
- processed_network.csv
- anomaly_results.csv
- anomaly_graph.png
